# websekolah
## Websekolah (Gen Z Vibes)  Portal web SMK dengan tampilan neobrutal/glassmorphism, landing estetik untuk siswa/orang tua, plus admin dashboard untuk kelola data dan fitur unggulan **Import Excel siswa**.  ### 1. Kebutuhan  - PHP **8.1+** (XAMPP terbaru). - MySQL / MariaDB (bawaan XAMPP). - Composer (untuk install PhpSpreadsheet).
